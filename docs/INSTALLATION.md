# Installation Guide

## System Requirements

- **OS:** Windows 10 or Windows 11 (64-bit)
- **RAM:** 4 GB minimum, 8 GB recommended
- **Storage:** 500 MB free space
- **Display:** 1280x720 minimum resolution

## Installing GuestFill

### Standard Installation

1. Download the latest `GuestFill-Setup-vX.X.X.exe` from the releases page.
2. Double-click the installer.
3. Follow the installation wizard.
4. Launch GuestFill from the Start Menu or Desktop shortcut.

### Portable Version

1. Download the portable ZIP file from the releases page.
2. Extract the ZIP to any folder (e.g., `C:\GuestFill`).
3. Run `GuestFill.exe` from the extracted folder.

## After Installation

1. Launch GuestFill.
2. The Home screen shows two main options:
   - **Create Excel from Documents** — OCR workflow
   - **Import Excel to Fill Guest Info** — Auto-fill workflow

## Verifying Installation

1. Open GuestFill.
2. The app should show the Home screen.
3. Go to Settings and verify all sections load.

## Uninstalling

1. Go to Windows Settings → Apps → GuestFill.
2. Click **Uninstall**.
3. To remove all data, delete the `GuestFill` folder in `%APPDATA%`.

## Troubleshooting Installation

### App does not start

- Check Windows requirements (Windows 10/11 64-bit)
- Reinstall the app
- Check antivirus logs (may quarantine new software)

### OCR does not work

- Make sure the installation completed fully
- Check that `guestfill-ocr.exe` exists in the installation folder
- Reinstall if needed
