# Support Guide

## How to Get Help

1. Check `docs/TROUBLESHOOTING.md` for common issues.
2. Export a diagnostic report from Settings screen.
3. Contact support with the diagnostic report and a description of the issue.

## Support Checklist

Before contacting support:

- [ ] Have you restarted GuestFill?
- [ ] Have you checked the error message?
- [ ] Have you exported a diagnostic report?
- [ ] What version of GuestFill are you using?
- [ ] What Windows version?
- [ ] What type of documents are you processing?
- [ ] What hotel system are you filling?

## Providing Diagnostic Reports

1. Go to **Settings** → **Diagnostics**.
2. Click **Export Diagnostic Report**.
3. Share the generated file with support.

**Warning:** Diagnostic files may contain masked sample data. Review before sharing.

## Common Support Requests

### OCR Not Reading Images Well

- Ensure images are at least 800x600 pixels
- Avoid heavy glare or shadows
- Ensure MRZ area is visible and not cropped
- Try converting PDFs to images first

### Excel Import Fails

- Ensure the file is not open in another program
- Check that required columns exist (fullName, gender, documentType, etc.)
- Ensure the file is .xlsx format

### Auto-fill Not Working

- Verify the template matches your target system
- Check that field mappings are correct
- Test with a single field first
- Ensure browser extension is connected (for web fill)
